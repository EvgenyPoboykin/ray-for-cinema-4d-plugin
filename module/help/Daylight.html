Daylight

Rotation X / Rotation Y

The position of the sun along the X axis (horizontally, 360 degrees) and along the Y axis (vertically, 90 degrees).

Daylight Color

Color of the sun.

Sun Brightness

The setting is responsible for the power of the sun.

Shadow Type

Type of shadows from the sun. There are two types of shadows: Area and Raytraced (Hard).

Shadow Density

The intensity of the shadow.

Use Artistic Shadows

Use Artistic Shadows. The default color of the shadow depends on the color of the light source. The color of the shadow is calculated according to the basic law of color science "Contrast pair". A contrasting pair are pairs that lie opposite each other in a color circle.

Shadows Color

If you uncheck "Use Artistic Shadows" then the shadow color options are set by the user.

Shadow Saturation / Shadow Value

The parameter "Shadow Saturation" is responsible for the saturation of the shadow, and the "Shadow Value" parameter is responsible for the fact that your shadow will be dark (less value - closer to black, more value - closer to white).

Use Fog

Use Fog. With the "Fog" settings, you can create an airborne perspective effect.

Fog Strength

The power of the Fog.

Fog Distance

The distance at which the Fog becomes 100%, based on the "Fog Strength" parameter.

Lock Fog Color / Fog Color

If there is a checkmark "Lock Fog Color", then the color of the Fog depends on the color of the light source (sun). If the "Lock Fog Color" check box is not selected, then the Fog color is set by the user.

Fog Saturation

The saturation of the Fog.

Show Daylight In Viewport / Show Fog

Show Daylight In Viewport - show the sun in the viewport. Show Fog - show the Fog in the viewing window. If you do not use the Fog in the scene, then "Show Fog" is not active.
