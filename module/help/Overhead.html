Overhead Softbox

Light mode

Overhead has two profiles, it is a rectangular Overhead and a round Overhead.

Width / Height

Dimensions Overhead, height and length.

Light Falloff

Falloff of light.

Show Falloff

Show the light Falloff in the viewport.

Color Light

Light color Overhead.

Use Master Light / Master Light

Use Master Light - use Master Light. Put the main light in the "Master Light" field and the global light will calculate its color based on the indicated light source. This field works both with Ray's plug-in tools and with standard lighting sources in Cinema 4D.

Opposite Color

Use opposite Color.

Light Brightness

The power of light is Overhead.

Samples

Samples for light.

Control Plane Brightness / Brightness Reflection Plane

A plane that is reflected in objects. If there is a checkmark "Control Plane Brightness", then the parameters of the glare are set by the user (Brightness Reflection Plane), if not, the glare strength depends on the intensity of light.

Falloff

Modes of falloff of light. Three modes - Inverse Square (Physical), Linear (Linear) and None (No).

Shadow Type

Type of shadows from Softbox. There are four types of shadows: Area, Raytraced (Hard), Shadow Maps (Soft) and No Shadows.

Use Artistic Shadows

Use Artistic Shadows. The default color of the shadow depends on the color of the light source. The color of the shadow is calculated according to the basic law of color science "Contrast pair". A contrasting pair are pairs that lie opposite each other in a color circle.

Shadows Color

If you uncheck "Use Artistic Shadows" then the shadow color options are set by the user.

Shadow Saturation / Shadow Value

The parameter "Shadow Saturation" is responsible for the saturation of the shadow, and the "Shadow Value" parameter is responsible for the fact that your shadow will be dark (less value - closer to black, more value - closer to white).

Shadow Density

The intensity of the shadow.

Maximum Samples

Samples for Area shadows of light when you use the Standard rendering engine.

Seen By Camera / Seen By Transparency / Seen By Camera Ray

Overhead display options for the final rendering. Visibility / Invisibility for camera, for beams, for transparent objects.

Seen By Reflection

Reflector display in reflection.

Seen By Specular

Visibility / Invisibility for highlights.

Standart Light On

On / Off Overhead.

Visible In Editor

Overhead visibility in the preview window.

High Quality Light(Slow)

Increases the quality of reflection of the reflector and increases the time to render the final image.
